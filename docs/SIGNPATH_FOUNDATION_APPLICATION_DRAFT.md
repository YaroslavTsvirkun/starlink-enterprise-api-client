### Description

`StarlinkEnterprise.ApiClient is an open-source .NET 10 client library for the Starlink Enterprise API. It provides hand-written DTOs, hand-written Refit endpoint contracts, DI-friendly domain clients, and automated tests so .NET applications can integrate with Starlink Enterprise services without code generation in the public API.`
